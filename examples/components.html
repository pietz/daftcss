What is Daft CSS?

A semantic-first CSS framework with shadcn/ui aesthetics. No JavaScript required for most interactions.

Is it compatible with Pico?

Yes! Same HTML patterns and class names for easy migration.

What browsers are supported?

Modern browsers: Chrome 123+, Firefox 120+, Safari 18+.
